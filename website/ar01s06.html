6. Customization 2 : Keeping your OS up-to-date using CVS...

We are going to download sources. Beware ! You need some space to download them all ! Please verify you have enough disk space before proceding....

src : around 680Mo //TO UPDATEpkgsrc : around 130Mo (without any "work") //TO UPDATExsrc : around Mo //TO UPDATE

Since I am not aware of an easy way in NetBSD to download sources, I have written my own script, based on the one for OpenBSD by James Z.

Just download it, and put it under /root

Edit it to change the first parameter : release and cvsroot, to match what you are looking for.

Now get the source from CVS repository :

./sysupdate src

You can also get xsrc (usually not really needed) if you want it :

./sysupdate xsrc
